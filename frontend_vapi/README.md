# Oto VAPI Hackathon Frontend

## How to work

- setup

  create `.env`

  ```bash
  cp .env.example .env.local
  ```

  set set below value

  ```txt
  NEXT_PUBLIC_PRIVY_APP_ID=
  PRIVY_APP_SECRET=
  ```

- format

  ```bash
  pnpm run format
  ```

- install

  ```bash
  pnpm install
  ```

- build

  ```bash
  pnpm run build
  ```

- dev

  ```bash
  pnpm run dev
  ```
